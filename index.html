Page.IsPostBack属性

作用：IsPostBack属性用于检测页面是否已经回传给服务器端。
对于每次页面加载是都需要被初始化的控件，可以将其代码放在IsPostBack中，对于只需要加载一次的控件，可以将其代码放（！IsPostBack）中。

小栗子：


protected void Page_Load(object sender, EventArgs e)
   {
       if (!Page.IsPostBack) {
           //创建items集合
           ArrayList items = new ArrayList();//如果不引用using System.Collections;会报错
           items.Add("苹果");
           items.Add("香蕉");
           //绑定到DropDownList，数据绑定，这两个语句不可缺一
           DropDownList1.DataSource = items;
           DropDownList1.DataBind();
       }
   }


<form id="form1" runat="server">
        <div>
   <asp:DropDownList ID="DropDownList1" runat="server">
       </asp:DropDownList>
       <asp:Button ID="Button1" runat="server" Text="Button" />
   </div>
       <asp:Label ID="Label1" runat="server" Text="Label"></asp:Label>
   </form>

在该控件属性中的AutoPostBack=true为自动回复；

发表于 2017-02-20 | 分类于 国际金融 |

第一节． 货币

货币起源：贝壳，因此与钱有关的文字都是贝字旁；

货币流通：执行流通手段职能的货币必要量
货币流通速度：
V=QP/M
M=QP/V，M为实际货币发行量；
M=待销售商品数量（Q）*单位商品价格（P）/同名货币流通次数（V）

银行分类：

  1. 一类：国家性 中央银行
  2. 二类：政策性 农业银行，开发银行，进出口银行，不以盈利为目的
  3. 三类：商业性 全国性国有，全国性股份制，区域性，地域性，地方性

金融界老大：bank-bank
我国把办理与银行有关的大金融机构称为银行：

  1. 1897年.5.27 中国通商银行
  2. 1950：“户部银行”-“大清银行”-“中国银行”（1911）
    3.1694：英格兰银行标志西方现代

货币缺点：银行制度的建立

银行危机的原因：商业银行的支付困难

外汇分为：狭义和广义
信用工具：汇票，本票，支票
外汇定义：

  1. 外汇具有静态和动态
  2. 国际信用工具
  3. 两个前提条件（自由兑换，普遍接受）
  4. 国际汇兑：不直接利用现金，而利用支票，汇票；信汇与电汇等信用工具，已委托付款或转让债权的方式来结算国际之间债权
  5. 美元是国际货币
    外汇的一般形式：以外币表示的国外资产
    外汇的基本形式：本国在外国通汇银行的活期存款
    外汇的基本职能：充当国际结算的支付手段
    最本质的特点：可自由兑换性

外汇种类：
自由外汇：三项义务，自由的多边的国际结算制度
IMF（国际货币基金组织）
